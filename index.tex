\chapter{Data Collection and Sources}\label{data-collection-and-sources}

There are lot of data sources available online to get started our
analysis.

Here, we will mainly focus on the ball-by-ball data so that we can
achieve the insights from more granular level.

We will be using open source data from
\href{https://cricsheet.org}{cricsheet.org}.

\section{Data Description}\label{data-description}

The raw data will be downloaded as a zip file. After unzipping we should
have multiple .csv files that corresponds to each T20 International
matches. There will be another file with same match ids but a suffix
``\_info'' giving the information of match summary. Our first objective
will be to collate all the ball by ball data in a single file or a data
frame in R.
